# arfa-types

TypeScript types and JSX declarations for the Arfa framework.

## What it provides

- A global `JSX` declaration so TS/TSX uses your `h` and `Fragment` functions.
- Exported helper types:
  - `PropsWithChildren<P>`
  - `Component<P>` / `FC<P>`
  - `ComponentProps<T>`
  - `IntrinsicElements` (permissive)
